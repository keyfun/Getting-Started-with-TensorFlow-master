# Getting Started with TensorFlow

## Code Testing Specifications

### Software required (With version): 
- Python(3.7)
- TensorFlow(1.10.1)
- Bazel(0.3.0)
- Pip(18.0)
- MatPlotlib(1.5.1)
- Pandas(0.18.1)
- NumPy(1.14.5)

### OS required:
- MacOS High Sierra 10.13.6
